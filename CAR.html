PENJELASAN DATASET#

Data tersebut berupa garis besar empat jenis mobil berbeda yang diekstraksi dari video lalu lintas menggunakan informasi gerak. Citra tersebut dipetakan ke dalam seri 1-D (berdasarkan fitur gerak dan bentuk). Kendaraan-kendaraan tersebut diklasifikasikan ke dalam salah satu dari empat kelas: sedan, pikap, minivan, atau SUV.

Kolom

Arti / Makna

Train Size (60)

Jumlah data pelatihan (training set) adalah 60 sampel time series. Model dilatih menggunakan 60 data ini.

Test Size (60)

Jumlah data pengujian (test set) adalah 60 sampel time series. Setelah model dilatih, ia diuji pada 60 data ini.

Length (577)

Panjang setiap time series adalah 577 titik data (misalnya 577 waktu, 577 sensor reading). Artinya tiap sampel berupa urutan angka sepanjang 577.

Number of Classes (4)

Dataset memiliki 4 kelas yang berbeda. Artinya model akan melakukan klasifikasi 4 kategori. Contoh: 4 jenis gerakan, 4 tipe sinyal, 4 label aktivitas, dll.

Number of Dimensions (1)

Setiap titik pada time series hanya memiliki 1 fitur.

BUSINESS UNDERTANDING#

Konteks:

Proyek ini beroperasi dalam domain klasifikasi objek atau pengenalan pola, dalam hal ini, klasifikasi tipe kendaraan.

Permasalahan (Problem Statement):

Dalam industri otomotif, logistik, atau asuransi, seringkali diperlukan cara cepat dan akurat untuk mengidentifikasi dan mengkategorikan tipe kendaraan (Sedan, SUV, Pickup, Minivan, dll.) berdasarkan serangkaian atribut teknis (dimensi, bobot, spesifikasi mesin, dll.). Proses identifikasi manual dapat memakan waktu, rentan kesalahan, dan tidak efisien untuk volume data yang besar.

Kebutuhan Bisnis:

Menciptakan sistem yang otomatis, cepat, dan konsisten yang dapat menerima data atribut kendaraan sebagai input dan mengeluarkan tipe kendaraan yang benar sebagai output.

import pandas as pd
import numpy as np
from scipy.io import arff
from sklearn.preprocessing import StandardScaler
from sklearn.svm import SVC
from sklearn.metrics import accuracy_score, classification_report, confusion_matrix
import matplotlib.pyplot as plt
import seaborn as sns

# -------- fungsi load arff --------
def load_arff(filename):
    """
    Memuat data dari file ARFF ke dalam DataFrame, melakukan decoding byte
    ke string, dan memetakan label kelas.
    """
    data, meta = arff.loadarff(filename)
    df = pd.DataFrame(data)

    # Decode byte → string untuk kolom object
    for col in df.select_dtypes(['object']).columns:
        df[col] = df[col].str.decode('utf-8')

    df.columns = meta.names()

    # Mapping label kelas
    mapping = {
        '1': 'Sedan',
        '2': 'Pickup',
        '3': 'Minivan',
        '4': 'SUV'
    }

    label_raw = df.columns[-1]
    df["label"] = df[label_raw].astype(str).map(mapping)

    # Identifikasi kolom fitur (yang dimulai dengan 'att')
    features = [c for c in df.columns if c.startswith("att")]
    return df, features, "label"

Tujuan:

Mengubah data mentah dari format khusus (.arff) menjadi struktur yang dapat diolah (Pandas DataFrame dan NumPy array).

Aksi Kunci:

Fungsi load_arff mengurus pembacaan file ARFF, decoding nilai byte menjadi string, dan memetakan label numerik (‘1’, ‘2’, dll.) ke nama kelas yang dapat dimengerti (‘Sedan’, ‘Pickup’, dll.).

load_data.py#

Tujuan Utama: Mengubah file data mentah berformat ARFF menjadi DataFrame Pandas yang siap diproses.

  • Fungsi Kunci: Memuat data dari Car_TRAIN.arff dan Car_TEST.arff menggunakan scipy.io.arff.

  • Preprocessing Data: Mengkonversi data yang berbentuk byte-string menjadi string biasa.

  • Output: Mengembalikan DataFrame, daftar nama fitur, dan nama kolom target.

# load_data.py
import pandas as pd
from scipy.io import arff

# Nama file train & test
TRAIN_FILE_NAME = "Car_TRAIN.arff"
TEST_FILE_NAME = "Car_TEST.arff"

def load_arff_data(path):
    """
    Load file ARFF dan konversi ke DataFrame pandas.
    Mengembalikan: df, nama_label, fitur, raw
    """
    try:
        data, meta = arff.loadarff(path)
    except Exception as e:
        print(f"Error membaca ARFF {path}: {e}")
        return pd.DataFrame(), None, None, None

    df = pd.DataFrame(data)

    # Konversi byte-string -> str
    for col in df.columns:
        if df[col].dtype == object:
            df[col] = df[col].apply(lambda x: x.decode() if isinstance(x, bytes) else x)

    target = df.columns[-1]               # kolom terakhir = label
    features = df.columns[:-1].tolist()   # semua kecuali label

    return df, target, features, meta

model_svc.py (Pelatihan Model & Penyimpanan)#

Tujuan Utama: Melatih model klasifikasi Support Vector Classifier (SVC) dan menyimpannya agar bisa digunakan di aplikasi web.

  • Pemetaan Label: Mengubah label numerik (1, 2, 3, 4) menjadi nama mobil (Sedan, SUV, dll.).

  • Preprocessing Penting: Melakukan Normalisasi (Scaling) fitur menggunakan StandardScaler. Ini wajib untuk model SVC.

  • Pelatihan Model: Melatih model SVC (SVC(kernel=’rbf’)) menggunakan data yang sudah di-skala.

  • Penyimpanan (.pkl): Menyimpan model SVC, StandardScaler, dan nama-nama kolom ke dalam satu file bernama model_svc.pkl menggunakan pickle.

import pandas as pd
import pickle
from sklearn.preprocessing import StandardScaler
from sklearn.svm import SVC
from load_data import load_arff_data, TRAIN_FILE_NAME

# -------------------------------
# MAPPING LABEL
# -------------------------------
# Mengubah angka 1-4 menjadi nama mobil
label_map = {
    "1": "Sedan",
    "2": "Pickup",
    "3": "Minivan",
    "4": "SUV"
}

# -------------------------------
# LOAD DATA TRAINING
# -------------------------------
# Memuat dataset training dari file ARFF
df_train, target_col, feature_cols, _ = load_arff_data(TRAIN_FILE_NAME)

# -------------------------------
# KONVERSI LABEL KE STRING
# -------------------------------
# Ubah kolom target menjadi string sesuai label_map
df_train[target_col] = df_train[target_col].astype(str).map(label_map)

# -------------------------------
# PERSIAPAN FITUR & TARGET
# -------------------------------
# Ambil nilai fitur (X) dan target (y)
X = df_train[feature_cols].astype(float).values  # Fitur numerik
y = df_train[target_col].values                  # Target berupa string

# -------------------------------
# SCALING / PREPROCESSING FITUR
# -------------------------------
# Normalisasi fitur agar memiliki mean=0 dan std=1
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# -------------------------------
# TRAIN MODEL SVC
# -------------------------------
# Membuat model Support Vector Classifier dengan kernel RBF
svc = SVC(kernel='rbf', probability=True)
svc.fit(X_scaled, y)  # Training model dengan data yang sudah diskalakan

# -------------------------------
# SIMPAN MODEL + PREPROCESSING
# -------------------------------
# Semua komponen penting disimpan dalam satu dictionary
model_dict = {
    "model": svc,           # Model SVC terlatih
    "scaler": scaler,       # Scaler untuk preprocessing fitur
    "features": feature_cols,  # Nama-nama fitur
    "target": target_col,      # Nama kolom target
    "df_test": df_train        # Test dataset (sama dengan train di sini)
}

# Simpan dictionary ke file pickle
with open("model_svc.pkl", "wb") as f:
    pickle.dump(model_dict, f)

print("✅ DONE! Model siap digunakan")

app.py (Aplikasi Streamlit Interaktif)#

Tujuan Utama: Membuat antarmuka web yang menampilkan evaluasi model dan memungkinkan pengguna melakukan prediksi sampel mobil.

  • Pemuatan Awal: Membaca file model_svc.pkl dan mengekstrak model dan scaler yang tersimpan.

  • Evaluasi Model: Menampilkan akurasi dan Classification Report model berdasarkan data uji (df_test).

  • Prediksi Interaktif:

    • Memungkinkan pengguna memilih sampel data dari dropdown.

    • Mengambil fitur sampel, melakukan scaling dengan scaler yang tersimpan, dan memprediksi tipe mobil.

    • Menampilkan hasil prediksi model dan label asli sampel.

  • Visualisasi: Membuat grafik deret waktu menggunakan Altair untuk memvisualisasikan nilai-nilai fitur dari sampel mobil yang dipilih.

import streamlit as st
import pandas as pd
import altair as alt
import pickle
from sklearn.metrics import accuracy_score, classification_report

# -------------------------------
# LOAD MODEL
# -------------------------------
MODEL_PATH = "model_svc.pkl"

try:
    # Load model, scaler, fitur, target, dan dataset test
    model_data = pickle.load(open(MODEL_PATH, "rb"))
    model = model_data["model"]          # Model SVC
    scaler = model_data["scaler"]        # Scaler untuk normalisasi
    saved_features = model_data["features"]  # Nama-nama fitur
    saved_target = model_data["target"]      # Nama kolom target
    df_test = model_data["df_test"]      # Data test
    st.success("🟢 model_svc.pkl berhasil dimuat!")
except Exception as e:
    st.error(f"❌ Gagal load model: {e}")
    st.stop()

# -------------------------------
# APP TITLE
# -------------------------------
st.title("🚗 Prediksi Tipe Mobil")

# -------------------------------
# EVALUASI MODEL
# -------------------------------
st.header("📊 Evaluasi Model")

# Ambil fitur dan lakukan scaling
X_test_scaled = scaler.transform(df_test[saved_features])
y_test = df_test[saved_target].values

# Prediksi seluruh data test
y_pred = model.predict(X_test_scaled)

# Hitung akurasi
accuracy = (y_test == y_pred).mean() * 100
st.metric("Akurasi Data Uji", f"{accuracy:.2f}%")

# Confusion report / classification report dalam bentuk dataframe
report = pd.DataFrame(classification_report(y_test, y_pred, output_dict=True)).transpose()
st.dataframe(report)

# -------------------------------
# DATA TEST HEAD
# -------------------------------
st.header("📄 Data Uji (head)")
st.dataframe(df_test.head())  # Menampilkan 5 baris pertama dataset test

# -------------------------------
# PREDIKSI PER SAMPEL
# -------------------------------
st.header("1️⃣ Prediksi Sampel Mobil")

# Membagi layout: kolom kiri untuk select box, kolom kanan untuk hasil prediksi
col1, col2 = st.columns([1,2])

# Pilih sampel berdasarkan index
idx = col1.selectbox("Pilih ID Sampel", df_test.index.to_list())
selected = df_test.loc[idx]

# Ambil fitur sampel dan ubah ke array 2D
X_sample = selected[saved_features].astype(float).values.reshape(1, -1)

# Prediksi model untuk sampel ini
pred_label = model.predict(scaler.transform(X_sample))[0]

# Label asli dari dataset
true_label = selected[saved_target]

# Tampilkan hasil prediksi dan label asli
with col2:
    st.markdown(f"### Prediksi Model: 🚘 **{pred_label}**")
    st.markdown(f"### Label Asli: 🏷️ **{true_label}**")

# -------------------------------
# GRAFIK TIME SERIES
# -------------------------------
st.header("2️⃣ Grafik Deret Waktu Fitur Mobil")

# Input untuk memilih range fitur yang ingin ditampilkan
col3, col4 = st.columns(2)
start = col3.number_input("Mulai fitur ke-", 1, len(saved_features), 1)
end = col4.number_input("Sampai fitur ke-", 1, len(saved_features), len(saved_features))

# Jika range valid, buat chart
if start <= end:
    feats = saved_features[start-1:end]  # Ambil nama fitur sesuai range
    df_plot = pd.DataFrame({
        "Langkah Waktu": range(start, end + 1),
        "Nilai Fitur": selected[feats].astype(float).values
    })
    chart = alt.Chart(df_plot).mark_line(point=True).encode(
        x="Langkah Waktu",
        y="Nilai Fitur",
        tooltip=["Langkah Waktu", "Nilai Fitur"]
    ).properties(
        title="📈 Grafik Deret Waktu Fitur Mobil"
    ).interactive()

    st.altair_chart(chart, width='stretch')

# -------------------------------
# DETAIL DATA SAMPEL
# -------------------------------
st.header("3️⃣ Detail Data Sampel")

# Expander untuk melihat semua detail kolom sampel
with st.expander("Klik untuk melihat detail"):
    st.dataframe(pd.DataFrame(selected).T)  # Transpose agar kolom menjadi row
